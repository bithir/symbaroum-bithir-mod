The Bithir's mod pack for the Symbaroum system is both an example on how to mod the Symbaroum system as well as containing utilities that Bithir wants for his game.

What does the module contain

The module contains the following additional functionalities:

  • Customised abilities/items

  • Inspirational dice

  • Generate shadow

  • Macros

Customised abilities

There are a few customised abilities

Berserk @UUID[Item.TS0XiHdZiUF3JYEi]

This works as the normal berserk, except instead of getting +1d4 armor at Adept, you instead get a dice upgrade on your existing armor.

Pack hunter @UUID[Item.IAmBx7tFaYQuvtzG]{Pack hunter}

This ability is a way for character with attack and defense of accurate to take advantage of this attribute without feeling like they are losing out against other character that used it as dump attribute.

This does not yet automatically prompt for the amount of bonuses.

Uncanny Coordination @UUID[Item.6drsU3hI68e9haUV]{Uncanny Coordination}

This ability is a way for character with attack and defense of accurate to take advantage of this attribute without feeling like they are losing out against other character that used it as dump attribute.

Customised items

These are used to assist in populating generated NPCs/Monsters as per their various abilities

Inspirational dice

The inspirational dice can be used to make rolls for inspiration when creating adventures or scenarios for the players. These are aimed towards Symbaroum, but idea of this is taken from story cubes.

At this time, there are four sets of inspirational dice:

  • Location [dl]

  • Events [de]

  • Creatures [dc]

  • Rewards [dr]

To use them, you can either use the included macro, or just make the roll using the abbreviation, i.e. '/roll 1dl + 2de + 1dc + 3dr' will roll one location dice, two event dice, one creature dice and 3 reward dice.

There is a macro included, 'Inspiration Roll', which you can decide on how many of each die you can use. It also formats the dice somewhat nicer. See macro list below.

Generate Shadow

Allows generation of a PC/NPC/Mosnter shadow. The shadow colour is loosely based on the Ordo Magica shadow scheme.

The button for this appears on the character sheet menu-bar  , which can be configured in the module settings

Macros

Currently the mod contains 3 macros

NPC/Monster Generator @UUID[Macro.mNAW5Mrw77dBBlZp]{NPC/Monster Generator}

Can be used to generate random abominations, ambriand or barbarians of different types. It has a small rule engine to generate them, so it is not completely random. See the Bithir's mods github for the code and data that drives it. It generates the NPC/Monster in a folder in the actor catalogue.

It is important to note that this requires abilities, traits, mystical power and items (weapons/armors) to be imported in to the world. It will only make NPC/Monsters with abilities that exists in the world.

Inspirational Roll @UUID[Macro.rDWIgUw0CJh7lfJG]{Inspiration Roll}

This makes a suitable roll and formats it in a nicer chat message, with elaborations on what each die kinda represents. It also tries (it is subject to lag, so sometimes they will go away anyway) to disable automatic dice fading when dice-so-nice, to make the dice go away, just shake them or make another roll.

Thus spoke Aroaleta @UUID[Macro.HdMVGkccCHVyrLAC]{Thus spoke Aroaleta}

This puts a quote of Aroaleta in to the chat window, with reference for story/book and sections. Perhaps nice to let your players see a quote at times, or to give to the barbarian in the party to pull up and recite and times.
